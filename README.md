# Convert Money

Conversor de moedas.

## Run Locally

Clone o projeto:

```bash
  git clone https://github.com/alexandrebekor/webapp-convert-money
```

Acesse o diretório:

```bash
  cd webapp-convert-money
```

Instale as dependências:

```bash
  yarn install
```

Inicie o servidor de desenvolvimento:

```bash
  yarn dev
```

## Tech Stack

**Client:** EJS, Tailwind.css

**Server:** Node.js, Express.js, Jest
